## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

`yarn` installed on your local machine.

### Installation

1. Clone the repo

```sh
git clone https://github.com/TGlide/weather-app.git
cd weather-app
```

2. Install Yarn dependencies

```sh
yarn
```

## Usage

### Run App

```sh
yarn start
```
